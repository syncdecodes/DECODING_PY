PYTHON — "in" checks VALUES (inside lists, strings, sets, tuples) and KEYS (in dicts)

JAVASCRIPT — "in" checks KEYS/INDEXES, NOT values

Simple examples to memorize
✔ Python
"a" in "cat"               # True
"a" in ["a", "b"]          # True
"a" in {"a": 1}            # True

✔ JavaScript
"a" in "cat"               // false (string has indexes only)
 0 in "cat"                 // true (index 0 exists)
"a" in {a: 1}              // true
